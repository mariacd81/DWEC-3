Enunciado Ejecutar
1.- Realiza un programa que pida por pantalla el tamaño y los elementos de un array.
2.- Escribe un programa que dado un array lo ordena y devuelve otro sin elementos repetidos.
3.- Escribe un programa que genere un array de Lunes a Domingo y luego lo divida en laborales y festivos.
4.- Escribe un programa que dado un array devuelva la posición del valor 3.
Datos: [2,5,9,7,6,3,1,5,4,8,3,2,6,9,3,5,1,2,3]
5.- Escribe un programa que pida una posición y una palabra.
6.- Escribe un programa que pida por pantalla el tamaño de una matriz.
7.- Escribe un programa que multiplique dos matrices.
