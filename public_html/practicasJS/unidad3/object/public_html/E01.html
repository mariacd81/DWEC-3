Realiza un programa con cuatro rectángulos, con su base, su altura y color como propiedades. Y como métodos aumento su base y altura, descenso su base y altura y modifico el color para sus propiedades.

A mayores tendrá dos métodos:

-mostrar(): Mostrará las propiedades de objeto debajo del div.

-asignarAId(String id): Se utilizará para asignar elementos al div.

Cada uno de ellos se va a corresponder con el div que queráis.
